# 🚗 Parking Slot Management System using FCFS

This project simulates a **Parking Slot Management System** using the **First Come First Serve (FCFS)** scheduling algorithm. It assigns vehicles to parking slots based on their arrival order and manages a waiting queue when all slots are full.

---

## 📌 Project Overview

- **Scheduling Algorithm:** FCFS (First Come First Serve)
- **Language:** Python
- **Use Case:** Mosque or public space parking management
- **Queue Handling:** `collections.deque` used for parked vehicles and waiting queue

## 🛠 Features

- Park incoming vehicles in the first available slot
- Maintain a waiting queue when slots are full
- Automatically assign freed slots to waiting vehicles
- Display current parking and queue status
- Error handling for invalid inputs
